메인 화면(ProblemList.jsp)에서 시작합니다. pdf에서 나온 것처럼, list.do로 실행합니다.
'글쓰기'를 누르면 문제 출제 화면이 나옵니다. 문제를 작성하면 메인에서 list에 추가된 글을
보여줍니다.
문제 list에서 문제 이름을 누르면 해당 문제의 내용을 보여줍니다.
문제 내용 화면에서
	'문제 삭제'는 아직 구현하지 않았습니다.
	'문제 풀기'를 누르면 ide 화면(답안 작성 화면)으로 넘어갑니다.
	'제출 답안 목록 보기'를 누르면 답안 리스트로 넘어갑니다.
	'목록으로 돌아가기'를 누르면 문제 list로 돌아갑니다.

ide에서
	원하는 언어를 선택하고, 소스 코드를 입력한 후 제출하기를 누르면 정답 체크 후 답안
	리스트에 추가, 답안 리스트 화면으로 너어갑니다.

제출 답안 목록에서
	'문제 목록으로'를 누르면 문제 list로 돌아갑니다.
	'사용 언어'를 누르면 문제 코드를 보여주도록 할 예정입니다.

만약 문제 코드 출력 화면이 구현되면
	'답안 목록으로' 버튼을 구현할 것이며 해당 버튼은 답안 리스트 화면으로 이동하게 할
	것입니다.

사용자 판별이 필요한 몇몇 기능은 해당 기능 없이, 임의로 제 이름을 넣어놨습니다. 추후 수정이
필요합니다.

answerscount와 problemscount라는 테이블이 있는데, 행 하나 열 하나로, 각각 문제 수, 질문 수를
저장하는 데이터베이스입니다.
